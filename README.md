# Demo Container

## Get Started

### Visibly Unique Instances

Run container instances with visibly unique background color by setting the env variable `BG_COLOR` to a color like:

  - Gray
  - Red
  - Orange
  - Yellow
  - Teal
  - Blue
  - Indigo
  - Purple
  - Pink

  `podman run -d -p 7000:3000 -e BG_COLOR=“blue” <container>`
